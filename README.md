# E-Commerce Sales Dashboard

## Overview
Analyzed 17K+ e-commerce orders using SQL, Excel, and Power BI to identify revenue trends, customer purchasing behavior, and product performance.

## Tools Used
- SQL
- Excel
- Power BI

## Key Insights
- Total Revenue: 21.78M
- Total Orders: 17K
- Units Sold: 51K
- Mobile users contributed the highest share of purchases
